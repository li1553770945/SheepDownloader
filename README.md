# SheepDownloader

![](https://img.shields.io/badge/qt-6.2.3-brightgreen)

一款基于QT的文件下载器，计划支持多线程、断点续传
目前使用的IDM下载器，但是感觉界面不太符合个人习惯，而且是付费的，除非用破解版，因此想要开发自己的开源下载器。目前刚刚有一些简单想法，计划按照以下需求开发。如果有志同道合的朋友可以一起开发。

1.初级功能：

+ 输入http网址能下载保存文件。√

+ 能进行一些简单的设置。（例如默认下载目录） √

2.中级功能：

+ 支持断点续传功能。

+ 支持多线程功能。

+ 有一个简单的GUI。 √

3.高级功能：

+ 开发配套浏览器插件，支持相应浏览器下载请求。

+ 有一个可高度自定义的GUI。

+ 支持磁力链接、p2p功能。
